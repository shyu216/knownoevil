Shor算法是一种量子算法，用于分解大整数为素数的乘积。这是一个量子计算课程辅导员发布的英文笔记的翻译版本。

PKC 公钥密码学

在公钥密码学中，Alice发布一个公钥，Bob使用该公钥来加密消息。然后，Alice使用她的私钥来解密消息。这依赖于密码学的非对称性：使用公钥加密消息很容易，但在没有私钥的情况下解密消息却很困难。这又依赖于单向函数的存在。

单向函数

  • 单向函数是指那些正向计算容易，但反向计算困难的函数。例如，RSA公钥加密系统使用因数分解作为单向函数：将两个素数 ppp 和 qqq 相乘得到一个合数 NNN 很容易，但反过来，将一个合数 NNN 分解为因数 ppp 和 qqq 却在数学上非常困难。
  • 事实上，已知的最好的经典因数分解算法——数域筛法（number field sieve）需要 exp⁡(Θ(n1/3log⁡2/3n))\exp(\Theta(n^{1/3} \log^{2/3} n))exp(Θ(n1/3log2/3n)) 次操作，其中 n=⌈log⁡2N⌉n = \lceil \log_2 N \rceiln=⌈log2​N⌉，即 NNN 的比特数。
  • 对于密码学应用来说，单向函数的数学定义至关重要，即它们在平均情况下必须难以反转，而不仅仅是在最坏情况下难以反转。

量子力学因式分解

1994年，Peter Shor发明了一种量子算法，可以在多项式时间内分解整数。这仍然是量子计算最重要和最令人印象深刻的潜在应用之一。Shor的算法建立在之前的查询复杂性算法之上，并基于两个关键见解：

  • 量子傅里叶变换算法可以用来解决阶（和周期）查找的数学问题。
  • 因数分解问题可以归约为阶/周期查找问题。

从周期查找中分解

  • 并不是所有的因数分解问题都很难；我们可以立即发现一个偶数，并知道它的一个因数是2。实际上，选择难以分解的数字有特定的标准，但基本思路是选择两个大素数的乘积。
  • 一个通用的因数分解算法首先会检查是否有因数分解的捷径（这个数是偶数吗？这个数是 N=abN = a^bN=ab 的形式吗？），然后在最坏的情况下使用 Shor 的周期查找算法。由于我们旨在关注算法的量子部分，我们将直接跳到 NNN 是两个素数的乘积的情况。

示例：分解 N=15N = 15N=15

  1. 第一步是选择一个介于 111 和 N−1N - 1N−1 之间的随机数 aaa。我们可以选择 111111，它不是 NNN 的非平凡因数（如果是，则将 NNN 除以这个 aaa 并重新开始）。
  2. 接下来，我们对 a=11a = 11a=11 和 N=15N = 15N=15 进行 Shor 的阶数查找算法。记住，我们测量的相位将是 sr\frac{s}{r}rs​，其中

    armod  N=1a^r \mod N = 1 armodN=1

    并且 sss 是 000 到 r−1r - 1r−1 之间的一个随机整数。如果 rrr 不是偶数，我们不能继续，必须尝试不同的 aaa 值。
  3. 现在我们有了 rrr，我们可能可以用它来找到 NNN 的一个因数。因为：

    armod  N=1a^r \mod N = 1 armodN=1

    那么：

    (ar−1)mod  N=0(a^r - 1) \mod N = 0 (ar−1)modN=0

    这意味着 NNN 必须整除 ar−1a^r - 1ar−1。如果 rrr 也是偶数，那么我们可以写成：

    ar−1=(ar/2−1)(ar/2+1)a^r - 1 = (a^{r/2} - 1)(a^{r/2} + 1) ar−1=(ar/2−1)(ar/2+1)

    那么 NNN 和 ar/2−1a^{r/2} - 1ar/2−1 或 ar/2+1a^{r/2} + 1ar/2+1 的最大公约数很可能是 NNN 的一个适当因数。如果在最后一步中，我们通过连分数算法找到 r=2r = 2r=2。那么我们有：

    ar/2−1=112/2−1=10和ar/2+1=112/2+1=12a^{r/2} - 1 = 11^{2/2} - 1 = 10 \quad \text{和} \quad a^{r/2} + 1 = 11^{2/2} + 1 = 12 ar/2−1=112/2−1=10和ar/2+1=112/2+1=12

    10 和 15 的（非平凡）最大公约数是 5，12 和 15 的（非平凡）最大公约数是 3。然后我们可以验证 3×5=153 \times 5 = 153×5=15，表明我们已经成功地将 15 分解为 5 和 3。

我们知道阶数 rrr 是一个整数，直观地说，我们需要足够的精度来区分 1r\frac{1}{r}r1​ 与附近的可能值，包括 1r+1\frac{1}{r+1}r+11​ 和 1r−1\frac{1}{r-1}r−11​。我们需要关注的最接近 1r\frac{1}{r}r1​ 的数字是 1r+1\frac{1}{r+1}r+11​，这两个数字之间的距离是

1r−1r+1=1r(r+1)\frac{1}{r} - \frac{1}{r+1} = \frac{1}{r(r+1)} r1​−r+11​=r(r+1)1​

∣y2m−1r∣<12r(r+1)\left| \frac{y}{2^m} - \frac{1}{r} \right| < \frac{1}{2r(r+1)} ​2my​−r1​​<2r(r+1)1​

对于满足

取 m=2log⁡(N)+1m = 2 \log(N) + 1m=2log(N)+1 可以确保我们有很大的机会使用前面描述的方法获得具有这种精度的估计。（如果我们对成功概率的下限为 40% 感到满意，那么取 m=2log⁡(N)m = 2 \log(N)m=2log(N) 就足够了。）

其他特征向量/特征值对

正如我们刚才所看到的，如果我们有 MaM_aMa​ 的特征向量 ∣ψ1⟩|ψ_1⟩∣ψ1​⟩，我们将能够通过相位估计来了解 rrr，只要我们使用足够的控制量子比特来获得足够的精度来做到这一点。不幸的是，获得特征向量 ∣ψ1⟩|ψ_1⟩∣ψ1​⟩ 并不容易，所以我们需要找出如何继续。

假设我们像上面一样进行，只是用特征向量 ∣ψk⟩|ψ_k⟩∣ψk​⟩ 代替 ∣ψ1⟩|ψ_1⟩∣ψ1​⟩，对于我们选择考虑的任何 k∈{0,…,r−1}k \in \{0, \ldots, r - 1\}k∈{0,…,r−1}。我们从相位估计过程中得到的结果将是一个近似值

y2m≈kr\frac{y}{2^m} \approx \frac{k}{r} 2my​≈rk​

在假设我们不知道 kkk 或 rrr 的情况下，这可能会或可能不会允许我们识别 rrr。例如，如果 k=0k = 0k=0，我们将得到一个接近 0 的近似值 y/2my / 2^my/2m，这不幸地告诉我们什么都没有。然而，这是一个不寻常的情况；对于其他 kkk 值，我们至少能够了解一些关于 rrr 的信息。

我们可以使用一种称为连分数算法的算法将我们的近似值 y/2my / 2^my/2m 转换为附近的分数——如果近似值足够好，包括 k/rk / rk/r。我们不会在这里解释连分数算法。相反，这里是关于该算法的一个已知事实的陈述。

给定一个整数 N≥2N \geq 2N≥2 和一个实数 α∈(0,1)\alpha \in (0, 1)α∈(0,1)，最多有一个整数对 u,v∈{0,…,N−1}u, v \in \{0, \ldots, N - 1\}u,v∈{0,…,N−1} 满足 v≠0v \neq 0v=0 且 gcd⁡(u,v)=1\gcd(u, v) = 1gcd(u,v)=1，并且满足 ∣α−uv∣<12N2|\alpha - \frac{u}{v}| < \frac{1}{2N^2}∣α−vu​∣<2N21​。给定 α\alphaα 和 NNN，连分数算法找到 uuu 和 vvv（或者报告它们不存在）。

该算法可以实现为大小为 O((log⁡(N))3)O((\log(N))^3)O((log(N))3) 的布尔电路。

如果我们有一个非常接近的近似值 y2m\frac{y}{2^m}2my​ 到 rk\frac{r}{k}kr​，并且我们为 NNN 和 α=y2m\alpha = \frac{y}{2^m}α=2my​ 运行连分数算法，我们将得到 uuu 和 vvv，如事实中所述。仔细阅读该事实使我们得出结论：

uv=kr\frac{u}{v} = \frac{k}{r} vu​=rk​

因此，我们不一定能知道 kkk 和 rrr，我们只知道最简形式的 kr\frac{k}{r}rk​。

例如，正如我们已经注意到的，我们不会从 k=0k = 0k=0 中学到任何东西。但这是唯一一个发生这种情况的 kkk 值。当 kkk 非零时，它可能与 rrr 有公因数——但我们从连分数算法中得到的数 vvv 必须能整除 rrr。

这并不明显，但这是一个已知的事实，如果我们有能力学习 uuu 和 vvv，对于 u/v=k/ru/v = k/ru/v=k/r，其中 k∈{0,…,r−1}k \in \{0, \ldots, r - 1\}k∈{0,…,r−1} 是均匀随机选择的，那么我们很可能在几次采样后恢复 rrr。特别地，如果我们对 rrr 的猜测是我们观察到的所有 vvv 值的最小公倍数，那么我们很可能是正确的。一些 kkk 值不好，因为它们与 rrr 共享公因数，当我们学习 uuu 和 vvv 时，这些公因数对我们是隐藏的。但随机选择的 kkk 不太可能长时间隐藏 rrr 的因数，并且我们猜测 rrr 不正确的概率随着样本数量的增加而指数下降。

总成本

实现每个受控幺正操作 MakM_a^kMak​ 的成本是 O(n2)O(n^2)O(n2)。有 mmm 个受控幺正操作，因此受控幺正操作的总成本是 O(n3)O(n^3)O(n3)。此外，我们有 mmm 个 Hadamard 门（其成本为 O(n)O(n)O(n)），量子傅里叶变换的成本为 O(n2)O(n^2)O(n2)。因此，受控幺正操作的成本主导了整个过程的成本——因此总成本为 O(n3)O(n^3)O(n3)。

通过阶数查找进行因数分解

最后我们需要讨论的是如何通过解决阶数查找问题来帮助我们进行因数分解。这部分完全是经典的——它与量子计算没有特别的关系。

基本思路是，我们想要分解数字 NNN，并且我们可以递归地进行分解。具体来说，我们可以专注于分裂 NNN 的任务，这意味着找到任意两个整数 b,c≥2b, c \geq 2b,c≥2 使得 N=bcN = bcN=bc。如果 NNN 是素数，这是不可能的，但我们可以首先使用素性测试算法高效地测试 NNN 是否为素数，如果 NNN 不是素数，我们将尝试分裂它。一旦我们分裂了 NNN，我们可以简单地对 bbb 和 ccc 进行递归，直到我们所有的因数都是素数，并且我们获得 NNN 的素数分解。

分裂偶数很容易：我们只需输出 2 和 N/2N/2N/2。
分裂完全幂次也很容易，这意味着数字的形式为 N=sjN = s^jN=sj，其中整数 s,j≥2s, j \geq 2s,j≥2，只需近似根 N1/2,N1/3,N1/4N^{1/2}, N^{1/3}, N^{1/4}N1/2,N1/3,N1/4 等，并检查附近的整数作为 sss 的嫌疑人。我们不需要超过 log⁡(N)\log(N)log(N) 步，因为此时根会下降到 2 以下，不会揭示其他候选者。

我们可以做这两件事是很好的，因为阶数查找对偶数或素数幂没有帮助，其中数字 sss 恰好是素数。
如果 NNN 是奇数且不是素数幂，阶数查找允许我们分裂 NNN。

输入一个奇数、合数且不是素数幂的整数 NNN。迭代以下步骤：

  1. 随机选择 a∈{2,…,N−1}a \in \{2, \ldots, N - 1\}a∈{2,…,N−1}。
  2. 计算 d=gcd⁡(a,N)d = \gcd(a, N)d=gcd(a,N)。
  3. 如果 d>1d > 1d>1，则输出 b=db = db=d 和 c=N/dc = N/dc=N/d 并停止。否则继续下一步，知道 a∈ZN∗a \in Z^*_Na∈ZN∗​。
  4. 设 rrr 为 aaa 模 NNN 的阶数。（这里我们需要阶数查找。）
  5. 如果 rrr 是偶数：
    1. 计算 x=ar/2−1(modN)x = a^{r/2} - 1 \pmod{N}x=ar/2−1(modN)
    2. 计算 d=gcd⁡(x,N)d = \gcd(x, N)d=gcd(x,N)。
    3. 如果 d>1d > 1d>1，则输出 b=db = db=d 和 c=N/dc = N/dc=N/d 并停止。
  6. 如果到达此点，迭代未能找到 NNN 的因数。

该算法的迭代可能未能找到 NNN 的因数。具体来说，这在两种情况下发生：

  • aaa 模 NNN 的阶数是奇数。
  • aaa 模 NNN 的阶数是偶数且 gcd⁡(ar/2−1,N)=1\gcd(a^{r/2} - 1, N) = 1gcd(ar/2−1,N)=1。

使用基本数论可以证明，对于随机选择的 aaa，至少有 1/2 的概率这两种情况都不会发生。事实上，对于 NNN 的不同素因数的数量 mmm，概率最多为 2−(m−1)2^{-(m-1)}2−(m−1)。这就是为什么假设 NNN 不是素数幂很重要。假设 NNN 是奇数也是必要的，这就是为什么（简单的）NNN 是偶数的情况必须单独处理。

因此，如果我们重复该过程 ttt 次，每次随机选择 aaa，我们将以至少 1−2−t1 - 2^{-t}1−2−t 的概率成功分裂 NNN。

我们不会详细分析这个过程，但这里是基本思路。如果我们选择的 aaa 使得 aaa 模 NNN 的阶数 rrr 是偶数，那么考虑数字 ar/2−1(modN)a^{r/2} - 1 \pmod{N}ar/2−1(modN) 和 ar/2+1(modN)a^{r/2} + 1 \pmod{N}ar/2+1(modN) 是有意义的。

使用公式 Z2−1=(Z+1)(Z−1)Z^2 - 1 = (Z + 1)(Z - 1)Z2−1=(Z+1)(Z−1)，我们得出结论：

(ar/2−1)(ar/2+1)=ar−1(a^{r/2} - 1)(a^{r/2} + 1) = a^r - 1 (ar/2−1)(ar/2+1)=ar−1

根据阶数的定义，我们知道 ar≡1(modN)a^r \equiv 1 \pmod{N}ar≡1(modN)，这意味着 NNN 整除 ar−1a^r - 1ar−1。因此，NNN 整除数字 (ar/2−1)(ar/2+1)(a^{r/2} - 1)(a^{r/2} + 1)(ar/2−1)(ar/2+1)，这意味着 NNN 的每个素因数必须要么整除 ar/2−1a^{r/2} - 1ar/2−1，要么整除 ar/2+1a^{r/2} + 1ar/2+1（或两者都整除）。

对于随机选择的 aaa，我们很可能会有 NNN 的素因数同时整除这两个项，这使我们能够通过计算 gcd⁡\gcdgcd 来分裂 NNN。
