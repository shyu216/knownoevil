Dart 是 2011 年 Google 搞出来的，最初想替代 JavaScript，后来因为 Flutter 这个跨平台 UI 框架而名声大噪，现在成了移动应用开发的香饽饽。

Dart明星项目Top3：1. **Flutter** - 开源跨平台UI框架，用于开发App如闲鱼、Google Ads，热重载快、渲染性能好，不过打包体积大、调试时内存占用高；2. **Google Ads** - 闭源谷歌广告应用，性能稳定、用户体验好，不过功能相对单一；3. **闲鱼** - 闭源二手交易App，跨平台效率高，不过Android端偶有卡顿。

架构师选 Dart 的时候，心里肯定在想："这玩意儿和 Flutter 深度集成，移动开发简直不要太爽；空安全机制在编译时就防止空指针异常，再也不怕 NPE 了；JIT + AOT 双重编译，开发时热重载快如闪电，生产时运行效率飞起；强类型系统既有静态检查又有类型推导，写起来自由度超高；一套代码能跑在 iOS、Android、Web、桌面多种平台，简直是开发者的福音；语法现代简洁，看起来赏心悦目。虽然生态相对较小，主要集中在 Flutter 领域，市场份额有限，学习曲线有点陡，要同时掌握 Dart 和 Flutter，工具链也要适应，但架不住它方便啊！" 所以在移动应用开发、跨平台应用、前端开发、服务器端开发这些地方，Dart 简直是如虎添翼。

和 Dart 相关的语言有 Kotlin，Android 开发的官方语言；Swift，iOS 开发的官方语言；JavaScript，前端开发的主流语言；Flutter，基于 Dart 的跨平台 UI 框架。但 Dart 就是在跨平台能力上特别能打，一套代码走天下，特别适合构建高性能、美观的跨平台应用。

不过学 Dart 也有难点，生态系统相对较小，主要集中在 Flutter 相关领域；市场份额有限，相比其他移动开发语言使用范围较窄；学习曲线相对陡峭，需要同时掌握 Dart 和 Flutter；工具链熟悉度需要适应特有的开发工具和流程。

总的来说，Dart 就像一个多面手，一个人能当几个人用，随着 Flutter 的流行，它已经成为移动开发领域的重要语言，是构建高性能跨平台应用的理想选择。
