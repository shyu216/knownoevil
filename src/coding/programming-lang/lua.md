Lua 是 1993 年巴西里约热内卢天主教大学搞出来的，最初设计就是一种嵌入式脚本语言，用来扩展应用程序的功能，现在以轻量级和高性能著称，简直是游戏开发和嵌入式系统的小能手。

Lua明星项目Top3：1. **OpenResty** - 开源Nginx扩展，用于Web服务如 Kong API网关，性能极高、扩展丰富，不过学习曲线陡峭、调试困难；2. **Love2D** - 开源2D游戏引擎，用于开发独立游戏，简单易学、跨平台，不过功能有限、不适合3D游戏；3. **Roblox** - 闭源游戏平台，用于开发用户游戏如《奔跑吧兄弟》，用户基数大、社区活跃，不过闭源受限、脚本限制多。

架构师选 Lua 的时候，心里肯定在想："这玩意儿轻量级设计核心库只有约 200KB，嵌入到各种应用程序中超级方便；动态类型系统不用声明变量类型，写起来自由度超高；协程支持实现协作式多任务，处理异步操作不在话下；元表机制提供强大的面向对象和元编程能力，简直是黑科技；简洁的 C API 容易和 C/C++ 代码集成；垃圾回收自动管理内存，省心；可扩展性容易通过 C 扩展功能。虽然性能不如编译型语言，生态系统相对较小，第三方库不如其他语言丰富，没有原生多线程支持要通过协程实现并发，元表概念学习曲线较陡，但架不住它轻便啊！" 所以在游戏开发（游戏脚本、AI 逻辑、配置管理）、嵌入式系统（设备固件、物联网设备）、配置管理（应用程序配置、规则引擎）、Web 开发（Nginx 脚本、Web 应用扩展）、科学计算（数据处理、模拟仿真）这些地方，Lua 简直是如鱼得水。

和 Lua 相关的语言有 Python，同样是动态类型脚本语言，但生态系统更丰富；JavaScript，动态类型脚本语言，主要用于 Web 开发；Ruby，动态类型脚本语言，注重开发效率；C/C++，Lua 可以与这些语言无缝集成。但 Lua 就是在轻量级、高性能和易于嵌入上特别能打，特别是游戏行业，是最常用的脚本语言之一，被许多知名游戏引擎采用。

不过学 Lua 也有难点，性能限制，性能不如编译型语言；生态系统相对较小，第三方库不如其他语言丰富；多线程支持没有原生支持，需要通过协程实现并发；元表概念学习曲线较陡，概念有点抽象。

总的来说，Lua 就像一个轻巧的小精灵，虽然身材小，但能力强，特别是在游戏开发、嵌入式系统、配置管理这些领域，它就是那个不可或缺的小帮手，是嵌入式脚本语言的首选。

### 一个代码块

```lua
function main()
    print("Hello " .. "Dale" .. "!")
end

main()
```

运行只需要 `lua hello.lua`。
