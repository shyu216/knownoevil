国道的炼奶咖啡和力神炭烧是我的咖啡初印象。食无定味，适口者珍。

| 咖啡品种        | 基本介绍                                   | 本人印象 |
|-----------------|----------------------------------------|----------|
| Decaffeinato   | 脱咖啡因咖啡（残留量＜0.1%），风味较温和，酸苦均衡 |          |
| Forte          | 浓烈、深烘风格，苦味重，烟熏、焦糖感           |          |
| Leggero        | 轻盈、浅烘风格，酸度明亮，花果香             |          |
| Ristretto      | 超浓缩（水量减半，萃取时间短），高甜度，粘稠感     |          |
| Espresso       | 标准浓缩（7-9g粉，25-30ml，25-30秒），平衡的酸甜苦 |          |
| Lungo          | 长萃（水量加倍，萃取时间长），更苦，风味稀释       |          |
| Crema          | Espresso表面的金黄色油脂层，芳香，微苦，绵密口感 |          |
| Arabica        | 阿拉比卡豆（高品质），酸甜丰富，风味复杂         |          |
| Robusta        | 罗布斯塔豆（高咖啡因），苦味重，木质、谷物感       |          |
| Single Origin  | 单一产地咖啡（突出风土特色），清晰的地域风味       |          |
| Blend          | 混合豆（平衡风味），均衡，适合大众口味         | Blend43 |
| Cold Brew      | 冷萃（低温长时间萃取），低酸，甜感突出，顺滑      |          |
| Americano      | Espresso加水，稀释的Espresso，苦味柔和       |          |
| Macchiato      | Espresso加少量奶泡，浓郁奶香，保留咖啡冲击力     |          |
| Flat White     | 双份Ristretto加微奶泡（比拿铁更浓），咖啡主导     |          |

## 购买建议

- **咖啡豆**：推荐选择单品豆或意式拼配豆，适合手冲或意式机使用
- **磨豆机**：手摇磨豆机适合家用，价格30-200元不等，电动磨豆机更方便
- **手冲器具**：滴滤杯、手冲壶、电子秤是基础装备
- **意式设备**：摩卡壶适合家庭意式咖啡制作，价格100-300元
- **清洁工具**：咖啡粉渣桶、清洁刷等小工具提升使用体验
