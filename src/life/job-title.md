### 1. Entry-level / Graduate / Junior
- **入门级 / 应届生岗**
- 适合：刚毕业、0–1年经验
- 关键词：
  - Entry-level / Entry
  - Graduate / Graduate Program（应届生项目、管培类）
  - Junior（初级）
- 特点：有人带、培训多、不要求独立负责大项目

### 2. Staff / Intermediate / Mid
- **正式员工 / 中级**
- 适合：1–3年经验
- 能独立完成任务，开始负责模块
- 最常见：Software Engineer、Data Engineer、Staff Engineer

### 3. Senior
- **高级**
- 适合：3–5年+ 经验
- 独立负责模块/系统，带新人，做技术方案
- 比如：Senior Software Engineer、Senior Data Analyst

### 4. Lead / Principal
- **技术负责人 / 首席**
- 负责技术方向、架构、带人做项目

### 5. Manager
- **管理岗**
- 不只是写代码：管团队、排期、绩效、沟通业务
- Engineering Manager、Technical Manager

# 二、方向名词扫盲
### 1. Software
- **Software Engineer / Developer**
  软件工程师、写代码、做系统、做App/网站/后端服务
- 最通用、岗位最多

### 2. Data
- **Data Analyst**：数据分析，报表、业务分析
- **Data Engineer**：数据工程，搭数据管道、库表
- **Data Scientist**：数据科学，建模、机器学习

### 3. Cloud
- **Cloud Engineer**
  云平台：AWS、Azure、GCP
  搭服务器、存储、网络、部署服务

### 4. Network
- **Network Engineer**
  网络工程师：路由器、交换机、防火墙、网络架构

### 5. DevOps / SRE
- 开发+运维，负责自动化、部署、稳定性
