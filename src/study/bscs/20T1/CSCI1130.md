## 课程简介 (What is this course about?)

**CSCI1130 - Java编程入门** 是香港中文大学计算机科学与工程系为工程类学生开设的 Java 编程入门课程。本课程主要教授 Java 编程基础，培养学生的结构化编程能力和面向对象编程思维。

**课程定位**：
- 面向工程学院学生
- 培养面向对象编程思维
- 为后续软件工程、数据结构课程打下基础

**知识在计算机大厦中的位置**：
```
┌─────────────────────────────────────┐
│     软件工程、设计模式               │
├─────────────────────────────────────┤
│     数据结构与算法 (CSCI2100)         │
├─────────────────────────────────────┤
│  ← CSCI1130: Java面向对象编程 →      │
├─────────────────────────────────────┤
│     ESTR1002: C语言编程基础          │
└─────────────────────────────────────┘
```

**核心内容**：
- **Java 基础**：数据类型、控制结构、方法
- **面向对象**：类、对象、继承、多态
- **图形界面**：Swing 组件、事件处理

**未来发展方向**：
- 企业级开发：Spring、Hibernate 框架
- 移动开发：Android 应用
- 大数据：Hadoop、Spark 生态

## 📚 课程笔记 (Course Notes)

1. **Java编程基础 (Java Basics)**
   - Java开发环境配置 (NetBeans)
   - 基本数据类型与变量
   - 算术与逻辑运算符
   - 程序控制结构 (if-else, switch, loops)

2. **方法与函数 (Methods)**
   - 方法的定义与调用
   - 参数传递与返回值
   - 静态方法 (static methods)
   - 方法重载初步

3. **面向对象编程入门 (Introduction to OOP)**
   - 类的定义与对象的创建
   - 构造函数 (Constructors)
   - 对象的引用与内存管理

4. **数组与字符串 (Arrays and Strings)**
   - 一维数组与多维数组
   - 数组的基本操作
   - String类的使用

### Assignment Projects (课程作业)

**Assignment 1: Shaky Display (抖动显示)**
- 截止日期: 2020年9月14日
- 目标: 练习基本Java编程和图形显示
- 内容: 创建一个简单的抖动显示效果程序
- 评分重点: 代码结构、注释规范

**Assignment 2: Cash Point (现金取款机应用)**
- 截止日期: 2020年10月23日
- 目标: 构建一个实际的现金账户管理应用
- 功能要求:
  - 查看余额 (初始余额 HKD 109,700.00)
  - 存款功能 (每次1-20张HKD100钞票)
  - 取款功能 (支持港币和韩元，自动找零)
  - 货币兑换计算 (1HKD : 150KRW)
  - 取款时显示最少数量的钞票组合
- 评分重点: 代码规范、注释、缩进风格

**Assignment 3: Quadratic Quiz (二次方程测验游戏)**
- 截止日期: 2020年11月6日
- 目标: 练习类的定义和对象创建
- 题目类型:
  - 两个实数根 (如 x² - 3x + 2 = 0，根为1和2)
  - 一个实数根 (如 x² - 2x + 1 = 0，根为1)
  - 无实数根 (随机生成B、C满足判别式 < 0)
- 游戏流程: 3道试用题 + 6道随机测验题

**Assignment 4: Reversi (黑白棋游戏)**
- 截止日期: 2020年12月7日
- 目标: 练习二维数组处理、文件I/O、异常处理和继承
- 游戏规则: 采用现代Othello规则
- 基本要求:
  - 8x8棋盘
  - 黑方先手
  - 玩家将棋子放在空位来"捕获"对方棋子
  - 捕获发生在八个直线方向上
- 扩展功能 (ReversiOnFile子类):
  - 从文件加载游戏局面
  - 保存当前游戏局面到文件
  - 练习继承、方法重写和多态
- 技术要点:
  - Unicode UTF-8文件格式处理
  - 异常处理
  - 图形用户界面开发

### Difficulties and Key Points (难点解析)

1. **GUI编程与用户交互**: 使用JOptionPane进行输入输出对话框的处理
2. **边界条件处理**: 
   - 输入验证 (范围检查、null处理)
   - 余额不足时的错误提示
   - 用户取消操作的正确响应
3. **浮点数处理**: 货币计算涉及浮点数精度问题
4. **循环结构设计**: 主菜单需要持续显示直到用户选择退出
5. **面向对象设计**: 需要理解对象、类、构造函数之间的关系
