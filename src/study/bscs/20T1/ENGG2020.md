## 课程简介 (What is this course about?)

**ENGG2020 - 数字逻辑与系统** 是香港中文大学计算机科学与工程系开设的数字逻辑与系统课程。本课程涵盖数字电路设计的基础知识，是计算机工程专业的核心必修课。

**课程定位**：
- 面向工程学院学生
- 理解计算机硬件底层原理
- 为后续计算机体系结构课程打下基础

**知识在计算机大厦中的位置**：
```
┌─────────────────────────────────────┐
│     计算机体系结构 (CSCI3150)         │
├─────────────────────────────────────┤
│  ← ENGG2020: 数字逻辑与系统 →        │
├─────────────────────────────────────┤
│     电路基础、物理基础                │
└─────────────────────────────────────┘
```

**核心内容**：
- **组合逻辑**：布尔代数、卡诺图、译码器
- **时序逻辑**：触发器、寄存器、计数器
- **存储器**：RAM、ROM、可编程逻辑

**未来发展方向**：
- 数字系统设计：FPGA、ASIC
- 计算机体系结构：CPU 设计
- 嵌入式系统：微控制器应用

## 📚 课程笔记 (Course Notes)

1. **数字逻辑基础 (Basic Digital Logic)**
   - 数制与编码 (二进制、八进制、十六进制)
   - 布尔代数基础
   - 逻辑门 (AND, OR, NOT, NAND, NOR, XOR, XNOR)
   - 组合逻辑电路设计基础

2. **组合逻辑设计 (Combinational Logic Design)**
   - 卡诺图 (Karnaugh Map) 化简方法
   - 真值表与逻辑表达式转换
   - 多路复用器 (MUX) 与解复用器 (DEMUX)
   - 编码器与译码器
   - BCD到七段显示器译码器

3. **时序逻辑设计 (Sequential Logic Design)**
   - 锁存器 (Latches) vs 触发器 (Flip-Flops)
   - RS锁存器 (NAND型/NOR型)
   - 触发器 (Flip-Flops): D, JK, T, SR
   - 边沿触发 vs 电平触发
   - 主从触发器 (Master-Slave FF)
   - Setup Time和Hold Time
   - 状态机设计基础
   - 同步与异步电路

4. **寄存器和计数器 (Registers and Counters)**
   - 移位寄存器 (SIPO, SISO, PISO, PIPO)
   - 异步计数器 (Ripple Counter)
   - 同步计数器
   - 模N计数器
   - 环形计数器 (Ring Counter)
   - 扭环计数器 (Twisted Ring Counter)

5. **算术逻辑单元 (ALU)**
   - 加法器与减法器 (全加器、进位传播)
   - 乘法器基础
   - 比较器与逻辑单元

6. **存储器与可编程逻辑 (Memory & Programmable Logic)**
   - RAM与ROM结构
   - SRAM vs DRAM
   - 只读存储器编程 (PROM, EPROM, EEPROM)
   - 存储器扩展
   - 可编程逻辑器件简介

### Lab Experiments (实验内容)

**Lab 1: Logisim和基本逻辑门**
- 工具: Logisim电路仿真软件
- 目的: 熟悉Logisim环境，掌握基本逻辑门的使用
- 练习:
  - 构建AND、OR、NAND、NOR门电路
  - 填写真值表
  - 推导输出逻辑函数
- 提交要求: 将Logisim电路文件(.circ)和实验报告打包提交

**Lab 2: BCD到七段显示器译码器**
- 目标: 设计BCD到7段显示器的译码电路
- 要求:
  - 使用卡诺图化简逻辑函数
  - 实现数字0-4的显示
  - 最终电路只能使用最多4个NOR门

### Difficulties and Key Points (难点解析)

1. **卡诺图化简**: 需要熟练掌握2-4变量卡诺图的圈选技巧
2. **逻辑门转换**: 将与或式转换为纯NOR门或NAND门实现
3. **锁存器与触发器区别**: 理解电平敏感 vs 边沿敏感的区别
4. **时序电路分析**: 理解触发器的工作原理和时序关系
5. **Setup Time和Hold Time**: 理解数据建立时间和保持时间的要求
6. **计数器设计**: 理解异步计数器和同步计数器的区别
7. **状态机设计**: 状态编码和状态转换图的设计
8. **存储器扩展**: 理解地址总线和数据总线的扩展方法
