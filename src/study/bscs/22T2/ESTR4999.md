## 课程简介 (What is this course about?)

**ESTR4999 - 毕业论文 II** 是香港中文大学工程学院理科课程的毕业论文第二部分，与 ESTR4998（毕业论文 I）配合完成。这是本科阶段最重要的学术研究项目的完成阶段。

**课程定位**：
- 面向工程学院理科学生（ESTR 系列）
- 本科毕业的必修项目
- 培养学术研究能力

**核心内容**：
- **论文撰写**：完整论文（30-50 页）
- **实验研究**：数据分析、结果验证
- **论文答辩**：口头报告、问答

**未来发展方向**：
- 学术研究：攻读硕士、博士
- 工业界：研发工程师
- 创业：技术创业

## 📚 毕业论文流程

### 阶段一：研究执行 (Term 1 继续)

**核心任务**：
- 按照研究计划开展实验
- 收集和分析数据
- 迭代改进研究方法

### 阶段二：论文撰写 (ESTR4999 期间)

**核心任务**：
- 撰写完整论文（通常 30-50 页）
- 遵循学术论文格式
- 包含：摘要、引言、相关工作、方法、实验、结论、参考文献

**论文结构**：
| 章节 | 内容 |
|------|------|
| 摘要 | 研究问题、方法、结果、贡献 |
| 引言 | 研究背景、动机、目标 |
| 相关工作 | 文献综述 |
| 方法 | 研究设计、算法/系统 |
| 实验 | 实验设置、结果分析 |
| 结论 | 总结、局限、Future Work |
| 参考文献 | 引用格式 |

### 阶段三：论文答辩

**核心任务**：
- 准备答辩 PPT（15-20 分钟）
- 展示研究成果
- 回答委员会问题

## ⚠️ 常见问题与建议

### Q: 论文写作有什么技巧？

1. **尽早开始写作**：不要等到最后才开始写
2. **保持写作节奏**：每周写一部分
3. **多读优秀论文**：学习论文写作风格
4. **及时与导师沟通**：定期提交草稿

### Q: 答辩需要注意什么？

1. **控制时间**：严格控制在规定时间内
2. **突出贡献**：重点讲自己的创新点
3. **准备问答**：预想可能被问的问题
4. **熟读论文**：对自己论文内容了如指掌

### Q: 如果研究遇到困难怎么办？

1. **与导师沟通**：及时反馈问题
2. **调整研究计划**：必要时可以修改方向
3. **寻求帮助**：可以向学长学姐请教
4. **保持耐心**：研究过程中遇到困难是正常的
