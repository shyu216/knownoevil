## 课程简介 (What is this course about?)

**IERG4130 - 网络安全入门** 是香港中文大学信息工程系的网络安全入门课程，涵盖网络安全的基本概念、攻击与防御技术、加密算法等核心内容。

**课程定位**：
- 面向信息工程、计算机科学专业学生
- 网络安全的核心课程
- 为渗透测试、安全架构打下基础

**知识在计算机大厦中的位置**：
```
┌─────────────────────────────────────┐
│     渗透测试、安全架构               │
├─────────────────────────────────────┤
│  ← IERG4130: 网络安全 →              │
├─────────────────────────────────────┤
│     计算机网络、操作系统              │
└─────────────────────────────────────┘
```

**核心内容**：
- **安全基础**：CIA 三元组、威胁模型
- **密码学**：对称加密、非对称加密
- **网络攻击**：SQL 注入、XSS、DDoS

**未来发展方向**：
- 渗透测试：漏洞挖掘、红队演练
- 安全架构：企业安全、云安全
- 安全研究：漏洞分析、恶意软件分析

### Lecture 2: Basic Concepts and Tools

**核心内容**：
- **CIA 三元组**：
  - **Confidentiality（保密性）**：防止未授权访问
  - **Integrity（完整性）**：防止数据被篡改
  - **Availability（可用性）**：确保系统随时可用

- **其他安全概念**：
  - **Authenticity（真实性）**：验证身份
  - **Authorization（授权）**：分配权限
  - **Accountability（可追溯性）**：追踪行为

- **威胁模型**：
  - **攻击类型**：主动攻击 vs 被动攻击
  - **攻击者**：内部 vs 外部
  - **防御措施**：预防性 vs 检测性 vs 恢复性

**难点**：理解信息安全的基本原则和威胁模型的建立。

### Part I: Basic Concepts and Tools

**核心内容**：
- **CIA 三元组**（Confidentiality, Integrity, Availability）
- 信息安全的基本概念
- 威胁模型与攻击分类
- 安全策略与防御措施

**CIA 三元组详解**：
| 概念 | 说明 | 例子 |
|------|------|------|
| **Confidentiality** | 保密性，防止未授权访问 | 密码、成绩、病历 |
| **Integrity** | 完整性，防止数据被篡改 | 邮件内容、金融数据 |
| **Availability** | 可用性，确保系统随时可用 | 服务器正常运行 |

**难点**：理解信息安全的基本原则和威胁模型的建立。

---

### Part II: Software Security

**核心内容**：
- **Buffer Overflow（缓冲区溢出）** 与防御
- **恶意软件（Malware）**
- 安全编码实践
- 软件漏洞分析

**难点**：缓冲区溢出的原理与利用方式，以及如何编写安全的代码。

---

### Part III: Cryptographic Algorithms

**核心内容**：
- **对称加密**（Symmetric Encryption）
- **非对称加密**（Asymmetric Encryption）
- 流密码 vs 分组密码
- **哈希函数**（Hash）
- **消息认证码**（MAC）
- **数字签名**与**证书**

**加密算法对比**：
| 类型 | 特点 | 例子 |
|------|------|------|
| 对称加密 | 速度快，密钥分发难 | AES, DES |
| 非对称加密 | 速度慢，密钥分发易 | RSA, ECC |
| 哈希函数 | 单向，不可逆 | SHA-256, MD5 |

**难点**：理解不同加密算法的适用场景，以及密钥管理的重要性。

---

### Part IV: Network Security

**核心内容**：
- 网络安全基础概念
- 典型攻击手段
- **防火墙**（Firewall）
- **入侵检测系统**（IDS）
- **DMZ（非军事区）**
- **Web 安全**：
  - Same-Origin-Policy（同源策略）
  - XSS（跨站脚本攻击）
  - CSRF（跨站请求伪造）
  - SQL 注入
- **无线安全**：WEP/WPA/WPA2
- **安全协议**：IPSec, TLS, PGP

**难点**：
- 理解各种 Web 攻击的原理和防御方法
- 防火墙与 IDS 的配置与使用

---

### Part V: System Security

**核心内容**：
- 访问控制
- 对手模型
- 移动安全
- 云安全基础

## ⚠️ 课程难点总结

1. **密码学基础**：对称/非对称加密、哈希函数、数字签名
2. **Web 安全漏洞**：XSS、CSRF、SQL 注入的理解与防御
3. **网络攻击与防御**：防火墙配置、IDS/IPS 原理
4. **综合应用**：如何将各种安全技术整合到实际系统中
5. **安全思维**：理解攻击者的思路，才能更好地防御

## 🔐 为什么网络安全很重要？

- **国家层面**：网络空间是继陆、海、空、天之后的"第五战场"
- **企业层面**：经济利益和声誉损失
- **个人层面**：隐私保护、数字资产安全

**攻击动机**：
- 为了乐趣（早期黑客）
- 为了利益（现代职业黑客）
- 为了国家安全（网络战）
