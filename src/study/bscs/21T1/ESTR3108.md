## 课程简介 (What is this course about?)

**ESTR3108 - 人工智能基础** 是人工智能领域的入门课程，介绍 AI 的核心概念、方法和应用。课程涵盖搜索、推理、学习等 AI 基础技术。

**课程定位**：
- 面向工程学院理科学生（ESTR 系列）
- AI 领域的入门课程
- 为机器学习、深度学习打下基础

**知识在计算机大厦中的位置**：
```
┌─────────────────────────────────────┐
│     深度学习、强化学习               │
├─────────────────────────────────────┤
│  ← ESTR3108: 人工智能基础 →          │
├─────────────────────────────────────┤
│     算法、概率论、线性代数            │
└─────────────────────────────────────┘
```

**核心内容**：
- **搜索算法**：A*、启发式搜索
- **知识表示**：逻辑推理、贝叶斯网络
- **机器学习**：监督学习、无监督学习
- **神经网络**：深度学习基础

**未来发展方向**：
- 深度学习：CNN、RNN、Transformer
- 强化学习：游戏 AI、机器人
- 自然语言处理：大语言模型

## 📚 课程笔记 (Course Notes)

#### 课程核心问题
- 如何让计算机"智能"地解决问题？
- 机器如何从数据中学习？
- 如何构建能够推理和决策的系统？

#### 课程主要内容

**第一部分：搜索与规划**
- **问题求解Agent**
  - 智能Agent的定义与结构
  - 环境的建模
- **无信息搜索策略**
  - 广度优先搜索（BFS）
  - 深度优先搜索（DFS）
  - 一致代价搜索（UCS）
- **有信息搜索（启发式搜索）**
  - 贪婪最佳优先搜索
  - **A*算法**：最优性证明与启发式函数设计
- **博弈与对抗搜索**
  - 极小极大算法（Minimax）
  - Alpha-Beta剪枝

**第二部分：知识表示与推理**
- **命题逻辑**
  - 语法与语义
  - 推理规则与证明方法
- **一阶逻辑**
  - 谓词、量词、推理
  - 逻辑编程简介
- **不确定性推理**
  - 概率基础
  - 贝叶斯网络
  - 隐马尔可夫模型（HMM）

**第三部分：机器学习基础**
- **监督学习**
  - 线性回归
  - 逻辑回归
  - 决策树
  - 支持向量机（SVM）
- **无监督学习**
  - K-means聚类
  - 主成分分析（PCA）
- **神经网络与深度学习基础**
  - 感知机与多层感知机
  - 反向传播算法
  - 卷积神经网络（CNN）简介
  - 循环神经网络（RNN）简介

**第四部分：自然语言处理基础**
- 词嵌入（Word Embedding）
- 语言模型
- 文本分类

**第五部分：强化学习基础**
- 马尔可夫决策过程（MDP）
- 值迭代与策略迭代
- Q-learning

#### 课程难点分析

1. **A*算法的最优性证明**
   - 理解可采纳性（Admissibility）和一致性（Consistency）
   - 设计有效的启发式函数

2. **贝叶斯推理**
   - 理解条件概率与贝叶斯定理
   - 构建和应用贝叶斯网络

3. **机器学习理论**
   - 偏差-方差权衡
   - 过拟合与欠拟合
   - 交叉验证

4. **神经网络训练**
   - 理解梯度下降与反向传播
   - 超参数调优
