课程简介 (What is this course about?)

ESTR1005 - 工程线性代数 是工程学生的核心数学课程，系统讲授矩阵、向量空间、特征值等核心概念。线性代数是现代科学计算的基石，广泛应用于机器学习、信号处理、量子力学等领域。

课程定位：

  • 面向工程学院理科学生（ESTR 系列）
  • 为后续数值分析、机器学习课程提供数学工具
  • 与 MATH1510（微积分）共同构成工程数学基础

知识在数学大厦中的位置：

核心内容：

  • 矩阵运算：高斯消元、行列式、逆矩阵
  • 向量空间：基、维数、线性变换
  • 特征值理论：对角化、Jordan 标准型
  • 有限域：编码理论、密码学应用

未来发展方向：

  • 数值线性代数：科学计算的核心
  • 机器学习：PCA、SVD、神经网络
  • 量子计算：量子态的线性代数描述
  • 应用领域：计算机图形学、信号处理、数据科学

Key Difficulties

  1. Understanding abstract vector space concepts: The transition from concrete vectors to abstract vector spaces can be challenging
  2. Rank-Nullity Theorem: Comprehending the relationship between rank and nullity
  3. Eigenvalue computations: Finding roots of characteristic polynomials, especially for larger matrices
  4. Finite fields: The conceptual shift from real/complex numbers to finite field arithmetic
  5. Multiple integrals: Setting up correct limits for double and triple integrals
  6. Jacobian transformations: Understanding the geometric meaning of the Jacobian determinant
  7. Line and surface integrals: Choosing the correct parametrization and understanding orientation

Additional Topics: Principal Components Analysis (PCA)

Principal Components Analysis (PCA) is a powerful statistical technique used for dimensionality reduction, pattern recognition, and data compression. It has found widespread applications in fields such as face recognition, image compression, and data analysis.

1. Statistical Background

1.1 Standard Deviation

  • Measures how spread out data is from the mean
  • Formula: σ = √[Σ(xᵢ - x̄)²/(n-1)]
  • Difference between sample and population standard deviation

1.2 Variance

  • The square of standard deviation
  • Measures the spread of data in a data set

1.3 Covariance

  • Measures the relationship between two dimensions
  • Positive covariance: dimensions increase together
  • Negative covariance: one dimension increases while the other decreases
  • Zero covariance: dimensions are independent

1.4 Covariance Matrix

  • For an n-dimensional data set, the covariance matrix is n×n
  • Diagonal elements: variances of each dimension
  • Off-diagonal elements: covariances between dimensions
  • Symmetric matrix

2. Eigenvectors and Eigenvalues

2.1 Eigenvectors

  • A vector that does not change direction when multiplied by a matrix
  • Only defined for square matrices
  • For an n×n matrix, there are n eigenvectors
  • Eigenvectors of a symmetric matrix are orthogonal (perpendicular)

2.2 Eigenvalues

  • The scalar factor by which the eigenvector is scaled
  • Always come in pairs with eigenvectors
  • Used to determine the significance of each principal component

3. PCA Method

Step 1: Collect Data

  • Obtain the data set with multiple dimensions

Step 2: Subtract the Mean

  • Calculate the mean of each dimension
  • Subtract the mean from each data point
  • This produces a data set with zero mean

Step 3: Calculate Covariance Matrix

  • Compute the covariance matrix of the mean-adjusted data
  • For 2D data: 2×2 matrix showing relationships between variables

Step 4: Calculate Eigenvectors and Eigenvalues

  • Find eigenvectors and eigenvalues of the covariance matrix
  • Order eigenvalues from largest to smallest
  • The eigenvector with the largest eigenvalue = principal component

Step 5: Form Feature Vector

  • Choose the number of components to keep
  • Form a matrix with selected eigenvectors as columns

Step 6: Transform Data

  • Multiply the feature vector by the mean-adjusted data
  • This gives the data expressed in terms of the principal components

4. Recovering Original Data

To reconstruct data from PCA transform:

  • Original Data ≈ Feature Vector × Transformed Data + Mean
  • Using fewer eigenvectors results in lossy compression
  • Information loss proportional to discarded eigenvalues

5. Applications

5.1 Face Recognition (Eigenfaces)

  • Represent face images as high-dimensional vectors
  • Apply PCA to find principal components (eigenfaces)
  • Compare new faces to known faces in eigenspace
  • Reduces computational complexity significantly

5.2 Image Compression

  • Represent images as vectors of pixel intensities
  • Keep only top k eigenvectors (k << original dimensions)
  • Achieves compression while preserving essential information
  • Used in JPEG, image databases

5.3 Data Analysis

  • Dimensionality reduction for visualization
  • Noise reduction
  • Feature extraction for machine learning
  • Financial portfolio optimization
